# Getting Started with ZyreHub Pro 🚀

Welcome to **ZyreHub Pro**, your ultimate developer assistant. This guide will help you set up the essential features in less than 2 minutes.

## 🔑 Configure your GitHub Token
To use the **Full Sync** and **Gist Sharing** features, you need to provide a GitHub Personal Access Token.
- Click the button below to open the settings.
- Paste your token in the `Zyrehub: Github Token` field.

[Configure Token](command:workbench.action.openSettings?%22zyrehub.githubToken%22)

## 🔥 Your First Sync
ZyreHub can automate your entire Git workflow. It detects if you have a repo, suggests a commit message, and even creates the repository on GitHub for you.
- Open any project folder.
- Run the **Sync Repository** command.

[Try Sync Repository](command:zyrehub.sync)

## 📋 Best Practices & Documentation
Keep your projects professional. Use the **Health Check** to find missing files and the **README Wizard** to generate beautiful documentation.
- Run the health check to see warnings.
- Use the wizard to generate your first README.

[Run Health Check](command:zyrehub.validate)
[Open README Wizard](command:zyrehub.readmeWizard)

## 📊 The Pro Dashboard
Get a high-level overview of your project stats and quick access to all tools from the central dashboard.

[Open Dashboard](command:zyrehub.openDashboard)
