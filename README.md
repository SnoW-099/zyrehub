# ZyreHub Extension

A Quality of Life (QoL) VS Code Extension for developers, featuring a minimalist dark theme and quick development tools.

## Features

- **ZyreHub Night Sky Theme**: A minimalist dark theme (black background with white and grayscale UI accents).
- **Fast Fix (`zyrehub.fastFix`)**: Automatically cleans trailing spaces and ensures trailing newlines in your current file.
- **Quick Commit (`zyrehub.quickCommit`)**: Rapidly auto-stages and prompts for a commit message, then executes the git commit gracefully.

## Installation

This is currently in development. You can run it locally by compiling the TypeScript code and pressing `F5` in VS Code to spawn an Extension Development Host.
