# Features - ZyreHub Pro

## 🚀 Git Automation (The "Hub")
- **Smart Sync**: Automatically detects repository state, stages files, generates a message, and pushes.
- **Smart Commit**: Uses logic to suggest meaningful messages based on file modifications.
- **Remote Creator**: If your local project doesn't have a GitHub remote, ZyreHub can create one for you instantly.

## 📊 Pro Dashboard
The central command center for your project. View file counts, line counts, and top languages at a glance. Access all ZyreHub tools via high-speed action buttons.

## 🛡️ Security & Quality
- **Security Audit**: Scans for leaked credentials (.env, keys) before you commit.
- **Health Check**: Ensures your project has the standard paperwork (README, license) and isn't bloated with large files.
- **Fast Fix**: One-click cleanup of trailing whitespaces.

## 📝 README Wizard
Stop writing boilerplate. Use the interactive wizard to generate a professional, structured `README.md` for your project in seconds.

## 📋 TODO Explorer
A dedicated panel in your sidebar that finds all `TODO`, `FIXME`, and `DEBUG` comments across your entire project.
